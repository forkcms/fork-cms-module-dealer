# Module: Dealer

The Dealer module allows users to search in a list of dealers and display the nearest in a Google Maps widget. Each dealer can have multiple brands and it's also possible to search directly on these brands.

## Installation

Visit the [Fork CMS knowledge base](http://fork-cms.com/knowledge-base) to learn how to install a module. To download the zip-package go to the [extension page](http://www.fork-cms.com/extensions/detail/dealer) of the module on fork-cms.com.
